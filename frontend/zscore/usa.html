Questions Fréquemment Posées (FAQ)

Visas de travail principaux :

  • H1B - Visa spécialisé (tech, finance) - 65,000 places/an + 20,000 Master US
  • L1 - Transfert interne entreprise (manager/spécialiste)
  • O1 - Talents exceptionnels (très difficile à obtenir)
  • E2 - Investisseur (création entreprise $100K+)
  • J1 - Échange temporaire (stage, recherche)
Conseil : Le H1B est le plus courant mais très compétitif (lottery system). Commencer par cibler les grandes entreprises tech qui sponsorisent.

Salaires recommandés par ville (célibataire) :

  • San Francisco : $120,000+ (coût vie élevé)
  • New York : $100,000+ (logement cher)
  • Los Angeles : $90,000+ (voiture obligatoire)
  • Seattle : $85,000+ (pas d'impôt état)
  • Austin : $75,000+ (tech hub émergent)
  • Dallas : $70,000+ (coût vie modéré)
  • Denver : $75,000+ (qualité de vie)
  • Miami : $65,000+ (pas d'impôt état)
Règle générale : Viser un salaire 2.5x à 3x le loyer mensuel pour être accepté par les propriétaires.

Système d'assurance privé obligatoire :

Via Employeur (recommandé)
  • Employeur paie 70-80% de la prime
  • Coût employé : $200-500/mois
  • Couverture familiale : +$800-1,500/mois
  • Deductible : $1,000-5,000/an
Assurance Individuelle
  • Marketplace (Obamacare)
  • Coût : $400-800/mois célibataire
  • Plans Bronze, Silver, Gold, Platinum
  • Subsidies selon revenus
Important : Sans assurance, une hospitalisation peut coûter $50,000+. L'assurance employeur est généralement la meilleure option.

Dépend totalement de la ville :

✅ Villes SANS voiture
  • New York : Métro excellent, parking cher
  • San Francisco : Transports publics corrects
  • Washington DC : Metro system efficace
  • Boston : Compact, marche possible
  • Chicago : CTA train/bus développé
❌ Villes AVEC voiture obligatoire
  • Los Angeles : Distances énormes
  • Austin : Transports publics limités
  • Dallas : Tout en voiture
  • Miami : Pas de metro développé
  • Phoenix : Ville étalée
Coûts voiture : $500-800/mois (lease + assurance + essence + parking). Alternative : Uber/Lyft quotidien peut revenir moins cher dans certaines villes.

Solutions pour nouveaux arrivants :

Options principales :
  • Cosigner américain : Ami/collègue garant
  • 6-12 mois loyer d'avance : Sécurité propriétaire
  • Bank statements : Preuves revenus/épargne
  • Employer letter : Confirmation emploi
  • Corporate housing : Temporaire 1-3 mois
Build credit rapidement :
  • Secured credit card : Deposit $500-1,000
  • Authorized user : Carte ami/conjoint
  • Credit builder loan : Banques locales
  • Pay bills on time : Rent, utilities
  • Keep low utilization : <30% limite carte
Timeline : Credit score 700+ possible en 6-12 mois avec discipline. Essential pour loans auto, mortgage, meilleures rates.
